## 🚀 Installation & Setup

### Option 1: Direct Upload
1. Create `images/` folder
2. Add all required images
3. Upload all HTML files and images folder to your web hosting
4. Access via your domain

### Option 2: GitHub Pages
1. Create new GitHub repository
2. Upload all files maintaining structure:
   ```
   repository/
   ├── index.html
   ├── products.html
   ├── custom-labels.html
   ├── about.html
   ├── serve.html
   ├── contact.html
   └── images/
       ├── [all images here]
   ```
3. Enable GitHub Pages in repository settings
4. Site will be live at `https://yourusername.github.io/repository-name/`

### Option 3: Local Testing
1. Create project folder
2. Add all HTML files
3. Create `images/` subfolder with images
4. Open `index.html` in any web browser
